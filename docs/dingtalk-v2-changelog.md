# 钉钉 v2.0 更新日志

## 版本信息

- **版本号**：v2.0
- **发布时间**：2026-03-12
- **变更类型**：功能增强（Feature Enhancement）

---

## 新增功能

### 1. 完整命令系统 ✨

新增 11 个命令，覆盖所有核心功能：

#### 基础命令
- `/帮助` `/help` — 显示所有可用命令
- `/状态` `/status` — 查看服务状态、活跃会话、定时任务
- `/终止` `/stop` — 停止运行中的任务

#### 会话管理
- `/新对话` `/new` — 归档当前会话，开始新对话
- `/会话` `/sessions` — 查看最近 10 个会话的列表
- `/会话 编号` — 切换到指定历史会话（支持编号或 ID 前缀）

#### 配置管理
- `/模型` `/model` — 查看/切换 AI 模型（支持编号、名称、模糊匹配）
- `/密钥` `/apikey` — 查看/更换 Cursor API Key

#### 任务管理
- `/任务` `/cron` — 查看所有定时任务及其状态
- `/任务 暂停 ID` — 暂停指定任务
- `/任务 恢复 ID` — 恢复指定任务
- `/任务 删除 ID` — 删除指定任务
- `/任务 执行 ID` — 手动触发任务执行

## 技术改进

### 1. 定时任务系统
- ✅ 集成 Scheduler 模块（与飞书共享）
- ✅ 独立配置文件 `cron-jobs-dingtalk.json`（与飞书隔离）
- ✅ Webhook 缓存机制（支持主动推送）
- ✅ 自动加载配置变更

### 2. 会话历史管理
- ✅ 支持查看最近 10 个会话
- ✅ 支持按编号或 ID 前缀切换会话
- ✅ 自动归档旧会话
- ✅ 显示会话创建时间和最后活跃时间

### 3. 模型切换
- ✅ 支持 7 个预置模型
- ✅ 编号快速选择（`/模型 1`）
- ✅ 名称模糊匹配（`/模型 opus`）
- ✅ 自定义模型名（列表外的模型）
- ✅ 实时生效（写入 .env 后 2 秒内自动加载）

### 4. 配置热更新
- ✅ API Key 可通过钉钉消息更换
- ✅ 模型可通过钉钉消息切换
- ✅ 无需重启服务

## 使用示例

### 快速上手
```
发送: /帮助
查看所有可用命令
```

### 切换模型
```
发送: /模型
选择: 1（切换到 Opus 4.6）
```

### 查看会话
```
发送: /会话
显示历史会话列表
```

### 创建定时任务
```
发送: 帮我创建一个定时任务，每天早上9点提醒我开会
AI 会自动创建并保存到 cron-jobs-dingtalk.json
```

### 管理任务
```
发送: /任务
查看所有任务

发送: /任务 暂停 abc123
暂停指定任务
```

## 升级指南

### 从 v1.0 升级到 v2.0

1. **停止旧版本服务**
   ```bash
   cd dingtalk-cursor-claw
   ./service.sh stop
   ```

2. **拉取最新代码**
   ```bash
   git pull
   ```

3. **启动新版本**
   ```bash
   ./service.sh start
   ```

4. **验证功能**
   - 在钉钉发送 `/帮助` 验证命令系统
   - 在钉钉发送 `/状态` 查看服务状态

### 配置迁移

无需额外配置，所有现有功能保持兼容：
- ✅ 会话数据自动继承（`.sessions.json`）
- ✅ 环境变量无需修改（`.env`）
- ✅ 定时任务独立配置（`cron-jobs-dingtalk.json`）

## 已知限制

### 1. 实时进度更新
- **限制**：钉钉不支持消息更新 API
- **影响**：无法显示 AI 思考/执行过程
- **方案**：只发送最终结果

### 2. 心跳检查
- **限制**：webhook 需要用户先发消息
- **影响**：无法主动唤醒用户
- **方案**：暂不实现，按需添加

### 3. 记忆搜索
- **限制**：需要向量嵌入服务
- **影响**：无法语义搜索历史记忆
- **方案**：按需实现
