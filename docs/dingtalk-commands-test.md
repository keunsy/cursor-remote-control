# 钉钉命令系统测试清单

## 测试方式

在钉钉中向机器人发送以下命令，验证功能是否正常。

---

## 1. 基础命令

### `/帮助`
**预期：** 显示所有可用命令列表
**验证点：**
- [ ] 显示基础指令
- [ ] 显示会话管理
- [ ] 显示模型与密钥
- [ ] 显示定时任务
- [ ] 显示项目路由

### `/状态`
**预期：** 显示服务状态信息
**验证点：**
- [ ] 显示 AppKey
- [ ] 显示调度器状态
- [ ] 显示活跃任务数
- [ ] 显示项目路由列表
- [ ] 显示活跃会话

---

## 2. 会话管理

### `/新对话`
**预期：** 归档当前会话，开启新对话
**验证点：**
- [ ] 显示"新会话已开始"
- [ ] 提示历史会话数量
- [ ] 下一条消息创建新会话

### `/会话`
**预期：** 显示历史会话列表
**验证点：**
- [ ] 显示工作区名称
- [ ] 列出最近 10 个会话
- [ ] 显示会话摘要和时间
- [ ] 标记当前会话
- [ ] 显示切换提示

### `/会话 2`
**预期：** 切换到第 2 个会话
**验证点：**
- [ ] 显示"已切换到会话 #2"
- [ ] 显示会话摘要
- [ ] 显示会话 ID 和时间
- [ ] 下一条消息在该会话中继续

---

## 3. 配置管理

### `/模型`
**预期：** 显示可用模型列表
**验证点：**
- [ ] 显示所有可用模型
- [ ] 标记当前模型（✅）
- [ ] 显示切换提示

### `/模型 1`
**预期：** 切换到第 1 个模型
**验证点：**
- [ ] 显示切换前后模型
- [ ] 显示"已写入 .env"
- [ ] 提示自动生效

### `/模型 opus`
**预期：** 模糊匹配切换模型
**验证点：**
- [ ] 匹配到正确模型
- [ ] 执行切换

### `/密钥`
**预期：** 显示当前 API Key（脱敏）
**验证点：**
- [ ] 显示 `...后8位`
- [ ] 显示用法提示

### `/密钥 key_test123`
**预期：** 更换 API Key
**验证点：**
- [ ] 显示"API Key 已更换"
- [ ] 显示新 Key 后 8 位
- [ ] 提示已写入 .env

### `/终止`
**预期：** 终止运行中的任务
**验证点：**
- [ ] 有任务时显示"已终止"
- [ ] 无任务时显示"无任务"
