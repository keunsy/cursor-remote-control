## 使用步骤

### 1. 配置钉钉应用

参考 `钉钉应用配置指南.md`：
- 创建企业内部应用
- 添加机器人能力（Stream 模式）
- 配置权限和事件订阅
- 获取 AppKey 和 AppSecret

### 2. 配置环境变量

```bash
cd ~/work/cursor/dingtalk-cursor-claw
vim .env
```

填入：
- `DINGTALK_APP_KEY`
- `DINGTALK_APP_SECRET`
- 其余配置可选

### 3. 测试运行

```bash
# 前台运行（调试）
bun run server-minimal.ts

# 看到以下输出说明成功：
# [钉钉] access_token 已刷新
# 钉钉 Stream 连接已建立，等待消息...
```

### 4. 发送测试消息

在钉钉中 @机器人：

```
@Cursor AI 你好
```

如果收到回复，说明配置成功！

### 5. 安装自启动（可选）

```bash
bash service.sh install
```

开机自动启动，崩溃自动恢复。
