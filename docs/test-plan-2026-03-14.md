## 测试用例

### 1. 工作区初始化

**目的：** 验证模板文件自动复制功能

**前置条件：** 删除或不存在 `.cursor/` 目录

**测试步骤：**

```bash
# 清空测试
rm -rf /Users/keunsy/work/cursor/cursor-remote-control/.cursor

cd /Users/keunsy/work/cursor/cursor-remote-control/dingtalk
bash service.sh restart

# 等待服务启动（约 5 秒）
sleep 5

# 检查日志
bash service.sh logs | grep "工作区"
```

**预期结果：**

- 创建 `.cursor/` 目录
- 复制所有模板文件（SOUL.md、IDENTITY.md、USER.md、MEMORY.md、HEARTBEAT.md、TASKS.md、BOOT.md、TOOLS.md、rules/*.mdc）
- 创建 `BOOTSTRAP.md`（首次启动）
- 日志中出现 `[工作区] 从模板复制: xxx`
- 日志中出现 `[工作区] xxx 初始化完成 (N 个文件)`

**验证命令：**

```bash
ls -la /Users/keunsy/work/cursor/cursor-remote-control/.cursor/
ls -la /Users/keunsy/work/cursor/cursor-remote-control/.cursor/BOOTSTRAP.md
ls /Users/keunsy/work/cursor/cursor-remote-control/.cursor/rules/
```

### 3. 启动自检

**目的：** 验证服务启动时自动执行 `.cursor/BOOT.md` 检查清单

**前置条件：** 完成测试 1 或 2，`.cursor/` 目录存在

**测试步骤：**

```bash
# 编辑 BOOT.md
cat > /Users/keunsy/work/cursor/cursor-remote-control/.cursor/BOOT.md << 'EOF'
# 启动自检

检查以下项目：
1. 环境变量配置
2. 项目路由
3. 记忆系统

如果都正常，回复 "HEARTBEAT_OK"。
EOF

# 重启服务
cd /Users/keunsy/work/cursor/cursor-remote-control/dingtalk
bash service.sh restart

# 等待自检执行（约 10 秒）
sleep 10

# 查看日志
bash service.sh logs | grep "启动"
```

**预期结果：**

- 日志显示 `[启动] 检测到 .cursor/BOOT.md，执行启动自检...`
- 日志显示 `[启动] .cursor/BOOT.md 自检完成`
- （可选）若 BOOT.md 返回非 HEARTBEAT_OK 内容，钉钉收到自检结果消息

### 4. 文件发送

**目的：** 验证 `/发送文件` 命令上传本地文件到钉钉

**前置条件：** 服务运行中，钉钉 webhook 已建立（需先发一条消息）

**测试步骤：**

```bash
# 创建测试文件
echo "测试文件内容" > /tmp/test-dingtalk.txt
```

在钉钉发送：`/发送文件 /tmp/test-dingtalk.txt`

**预期结果：**

1. 收到「📤 正在上传文件」或「上传中」提示
2. 收到文件（test-dingtalk.txt）
3. 收到「✅ 文件发送成功」或「发送成功」提示

**备选测试（~ 路径）：**

```
/发送文件 ~/Desktop/test.txt
```

**错误场景测试：**

- 发送 `/发送文件 /tmp/不存在.txt` → 预期：收到「文件不存在」提示
- 发送超过 30MB 文件 → 预期：收到「文件过大」提示

## 附录：快速回归脚本

以下命令可用于快速验证核心流程（不包含钉钉消息发送）：

```bash
#!/bin/bash
# 快速回归：工作区初始化 + 启动自检
ROOT="/Users/keunsy/work/cursor/cursor-remote-control"
DINGTALK="$ROOT/dingtalk"

echo "=== 1. 清空 .cursor ==="
rm -rf "$ROOT/.cursor"

echo "=== 2. 重启服务 ==="
cd "$DINGTALK" && bash service.sh restart
sleep 5

echo "=== 3. 检查工作区日志 ==="
bash service.sh logs | grep "工作区" | tail -20

echo "=== 4. 检查 .cursor 目录 ==="
ls -la "$ROOT/.cursor/" 2>/dev/null || echo "目录不存在"

echo "=== 5. 检查 BOOTSTRAP.md ==="
ls -la "$ROOT/.cursor/BOOTSTRAP.md" 2>/dev/null || echo "文件不存在（可能已删除）"
```
