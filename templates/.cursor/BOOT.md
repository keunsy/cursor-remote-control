# 启动自检清单

每次服务启动时自动执行。

## 检查项

1. 检查 `.env` 配置完整性
2. 检查项目路由配置
3. 检查记忆系统状态
4. 检查定时任务状态

## 输出规则

- 如果一切正常，只回复 "HEARTBEAT_OK"
- 如果发现问题，**必须在问题前加上标识**：
  - ✅ = 正常
  - ⚠️ = 警告（需要关注但不影响运行）
  - ❌ = 错误（影响功能，需要修复）
  - 🔔 = 建议配置（可选优化项）

## 输出示例

### 正常情况
```
HEARTBEAT_OK
```

### 发现问题
```
✅ 项目路由：已加载 3 个项目
⚠️ 定时任务：发现 1 个示例任务被启用（已自动禁用）
❌ .env 配置：缺少 VOLC_EMBEDDING_API_KEY，向量搜索不可用
🔔 记忆系统：建议运行索引以提升搜索性能
```
